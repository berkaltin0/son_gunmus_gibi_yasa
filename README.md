# Live Like It's Your Last Day<br><br>This website is a project that visualizes real-time changes in the world's population. The site allows you to live track birth and death events happening around the world at any given moment, reminding you of the value of every moment.<br><br>## Features<br><br>- Real-time birth counter<br>- Real-time death counter<br>- Live world population indicator<br>- Net population growth rate<br><br>## How it Works?<br><br>The data on the site is based on estimates derived from demographic research worldwide. Values ​​used:<br><br>- World population: Approximately 8 billion (2023 estimate)<br>- Average 4.3 births per second<br>- Average 1.8 deaths per second<br><br>These figures are based on statistical data from sources such as Worldometers and the United Nations.<br><br>## Installation<br><br>The project is entirely client-side and does not require any server. To run:<br><br>1. Download the project to your computer<br>2. Open the `index.html` file in any web browser.<br><br>## Technologies<br><br>- HTML5<br>- CSS3<br>- JavaScript (Vanilla)<br><br>## To-Dos and Improvements<br><br>- Connecting to a real API<br>- Adding statistics by country<br>- Adding comparison graphs with historical data<br>- Improving mobile experience<br><br>## License<br><br>This project is distributed under the MIT license.
